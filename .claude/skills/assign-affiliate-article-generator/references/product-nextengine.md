# Product Reference — Next Engine

`assign-affiliate-article-generator` が Next Engine の記事を書く際の事実拠り所。

## 製品概要

- 提供元: Hamee 株式会社 (https://next-engine.net/)
- カテゴリ: EC 一元管理 SaaS (OMS = Order Management System)
- ターゲット: 中小〜中堅 EC 事業者 (年商 1〜100 億円規模)
- リリース年: 2008 年〜 (国内シェアトップクラス)
- 公式 X / Twitter: @next_engine (https://x.com/next_engine)

## 主要機能

| カテゴリ | 機能 |
|---|---|
| 受注一元管理 | Amazon / 楽天 / Yahoo! / Wowma / Qoo10 / au PAY マーケット / Shopify / カラーミー等の受注をまとめて取り込み |
| 在庫一元管理 | 全モール在庫を自動同期 (差分更新・販売数自動反映) |
| 商品マスタ管理 | 商品データを一元管理、各モールに反映 (項目マッピングあり) |
| 出荷指示 | ピッキングリスト、送り状連携 (ヤマト B2 / 佐川 e-飛伝 / 日本郵便) |
| メール自動化 | 注文確認・発送通知メールテンプレートで自動送信 |
| アプリストア | サードパーティ製の拡張機能 (LINE 連携、AI レコメンド、自動翻訳など) |
| API | REST API 公開 (受注 / 在庫 / 商品マスタ / メール) |

## USP (差別化要素)

1. **国内モール対応の網羅性**: 主要な国内 EC モールの 95% 以上をカバー
2. **アプリストアの拡張性**: 100 以上の拡張アプリで業務にフィット
3. **API の柔軟性**: バックオフィスシステム連携が容易
4. **国内シェアトップクラス**: 11,000 社以上の導入実績、ノウハウ豊富
5. **送り状自動連携**: 配送業者 API と直接統合

## 競合

| 競合 | 強み | Next Engine との違い |
|---|---|---|
| CROSS MALL | 海外モール (eBay, Amazon US 等) 対応が強い | Next Engine は国内モール特化 |
| アイテムマトリックス (iMatrix) | 在庫管理特化、シンプル | Next Engine は受注・在庫・出荷を統合 |
| TEMA POS | リアル店舗との在庫統合 | Next Engine はオンライン特化 |
| BASE / STORES | EC モール自体 (OMS ではない) | レイヤーが違う |

## 弊社 (合同会社スマイルコンフォート) の導入支援

**Next Engine 導入支援は弊社の主力サービスの 1 つ**。実績ベースで以下を提供:

- 初期セットアップ (アカウント開設・モール接続・在庫マッピング)
- 商品マスタの移行 (既存システムからの移管)
- API カスタマイズ (バックオフィス・基幹システム連携)
- アプリストア活用提案
- 運用フロー設計と社内オペレーション定着支援
- トラブル対応 (受注同期エラー、在庫齟齬等)

加えて、楽天市場 / Yahoo! ショッピングのカテゴリ・属性マッピングを AI で補完する自社プロダクト **CataMap** (https://catamap.app/) を運用しており、Next Engine 周辺の商品マスタ整備や複数モール出品の前処理にも応用できる。
