「ネクストエンジンの料金が分かりづらい」というご相談を毎月頂きます。結論から言えば 2026 年 5 月時点の [ネクストエンジン](https://base.next-engine.org/account/?agent_code=MzEzNw) の公式料金は **月額 3,000 円 (税抜) + 受注件数の階層別従量課金** で、初期費用は 0 円です[^1]。本記事では公式の料金ページに記載された数字をベースに、受注ボリュームごとの月額シミュレーションと見落としがちな年間保守費用まで、弊社の Next Engine 導入支援で見てきた実運用目線で整理します。

## ネクストエンジンの公式料金構造を 1 枚で把握する

[ネクストエンジン](https://base.next-engine.org/account/?agent_code=MzEzNw) の課金は「基本料金」「受注件数の従量課金」「オプション (有料アプリ・外部連携)」「年間保守費用」の 4 要素で決まります[^1]。

### 公式の課金 4 要素

- **基本料金**: 月額 **3,000 円 (税抜)**。月間受注 200 件まで含む
- **受注件数の従量課金**: 201 件目から階層別 (35 円→5 円/件 まで逓減)
- **初期費用**: **0 円** (店舗数追加・商品登録数追加も 0 円)
- **年間保守費用**: 契約から 1 年経過後、毎年契約月に **15,000 円 (税抜)**

「受注件数」はモール別ではなく **全モール合算** でカウントされます[^1]。Amazon・楽天・Yahoo! ショッピング・Shopify を併用する事業者は 200 件の閾値を想定より早く超えるので、複数モール出店している場合の試算では合算で計算してください。

### 受注件数の階層別単価 (公式)

公式の料金ページに明示されている階層別単価は以下のとおりです[^1]。すべて税抜表示で、201 件目から段階的に逓減します。

| 受注件数の階層 | 1 件あたりの単価 |
|---|---|
| 1〜200 件 | (基本料金に含む) |
| 201〜400 件 | 35 円 |
| 401〜1,000 件 | 30 円 |
| 1,001〜3,000 件 | 25 円 |
| 3,001〜5,000 件 | 20 円 |
| 5,001〜7,000 件 | 15 円 |
| 7,001〜10,000 件 | 10 円 |
| 10,001 件〜 | 5 円 |

階層を超えるごとに逓減するため、ボリュームが増えるほど 1 件あたりの限界費用は下がります。

## 受注ボリューム別の月額シミュレーション (公式単価から積算)

公式の階層別単価から、月間受注件数ごとの月額を積み上げ計算した結果が以下です。**すべて税抜・基本料金 3,000 円込み・有料アプリ等のオプションは別**です。

| 月間受注件数 | 月額 (税抜) | 内訳 |
|---|---|---|
| 200 件まで | **3,000 円** | 基本料金のみ |
| 400 件 | **10,000 円** | 3,000 + 200 件 × 35 円 |
| 1,000 件 | **28,000 円** | 上記 + 600 件 × 30 円 |
| 3,000 件 | **78,000 円** | 上記 + 2,000 件 × 25 円 |
| 5,000 件 | **118,000 円** | 上記 + 2,000 件 × 20 円 |
| 7,000 件 | **148,000 円** | 上記 + 2,000 件 × 15 円 |
| 10,000 件 | **178,000 円** | 上記 + 3,000 件 × 10 円 |

弊社スマイルコンフォートは [ネクストエンジン](https://base.next-engine.org/account/?agent_code=MzEzNw) の導入支援で多数の事業者をサポートしてきましたが、複数モール合算で 1,000〜3,000 件レンジに着地する事業者が最多です。試算でこのレンジに入る見込みなら、月額は 28,000〜78,000 円 (基本 + 従量) が目安になります。

## 無料体験と初期費用

新規契約のハードルを下げる仕組みとして、以下が公式に用意されています[^1]。

### 公式の無料体験

- **無料体験期間: 30 日間**
- 主要機能 (受注処理・在庫同期・出荷管理・商品マスタ管理) を本番に近い形で検証可能
- 体験期間中に発生する受注件数も従量課金に含まれない

### 初期費用 0 円とそのインパクト

公式に **初期費用 0 円** が明示されているため、サインアップから本契約までの初期キャッシュアウトは月額 3,000 円のみ。検証用に短期で契約し、自社の受注フローと相性を見たうえで本格運用に移すことができます。

無料体験期間中に **自社の実勢受注ボリューム** で走らせると、本契約後の月額レンジが ±20% 程度の精度で見えるようになります。

## 見落としがちな追加コストと最適化ポイント

公式料金ページに明記はあるものの、初回契約時に見落とされやすい 4 つの追加コストを整理します[^1]。

### 1. 年間保守費用 15,000 円

契約から 1 年経過後の毎年、契約月に **年間保守費用 15,000 円 (税抜)** が請求されます。初年度は発生しないため、2 年目以降の年予算に組み込み忘れがちです。月割すると月額換算で約 1,250 円の上乗せです。

### 2. 有料アプリ (アプリストア)

ネクストエンジンのアプリストアでは LINE 連携・AI レコメンド・自動翻訳など 100 以上の拡張アプリが提供されており、個別に月額料金が発生します。アプリごとに価格は異なるため、必要なものに絞って導入することで月額の膨張を防げます。

### 3. 外部システム連携の料金

公式の料金ページには外部システム連携の料金例も掲載されています[^1]:

- **POS レジ**: 0〜7,000 円
- **CRM・顧客管理**: 0〜20,000 円
- **倉庫・WMS**: 0 円〜従量課金制
- **Amazon FBA マルチチャネル**: 7,000 円〜従量課金制

実店舗 + EC のオムニチャネル運用や、外部倉庫を使う場合はこれらが加わります。

### 4. API カスタマイズの開発工数

API でバックオフィスや基幹システムと連携する場合、開発工数は社内かベンダーで賄います。弊社のような外部支援を使う場合は、初期スコープにより数十万円〜の規模で MVP を組むケースが多いです。

## 自社受注規模に合った見積もりを取る手順

実勢価格を抑えたら、最後は **自社条件で正式見積もりを取る** ステップです。30 日の無料体験から逆算するのが最短ルートです。

1. [ネクストエンジン](https://base.next-engine.org/account/?agent_code=MzEzNw) の公式サイトにアクセスし、無料体験を申し込む
2. 申込フォームに **想定月間受注件数・連携モール・出荷フロー** を記入
3. 営業担当との初回ミーティングで **2 年目以降の年間保守費用** や **外部システム連携の有無** を確認
4. **オプション (有料アプリ・外部連携)** の試算を貰う
5. 無料体験中に **本番想定の受注ボリューム** で走らせ、課金後の月額を予測

この手順で見積もりレンジが圧縮され、本契約後に「想定より高かった」事態を避けられます。
