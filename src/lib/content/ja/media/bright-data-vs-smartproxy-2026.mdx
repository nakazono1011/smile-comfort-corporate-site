「Bright Data と Smartproxy、結局どちらを選べばいいのか」は、2026 年に入ってからも問い合わせが絶えないテーマです。本記事では、Smartproxy が 2024 年に Decodo へリブランドした事実関係を整理した上で、料金単価モデル・地理カバレッジ・日本市場サポート・SLA・コンプライアンスの 5 軸を 2026 年 5 月時点の最新情報で比較し、チーム規模と用途別の選定フローを実運用目線で提示します。弊社が Bright Data を本番運用してきた経験と、Smartproxy / Decodo を比較検証した実測値をもとに整理します[^1]。

## Smartproxy と Decodo - リブランドの整理

最初に混乱しやすい点を整理します。**Smartproxy は 2024 年に Decodo へリブランドされた同じサービス** であり、別会社ではありません。公式サイトのドメインと製品名はすでに Decodo へ移行していますが、市場では旧 Smartproxy 名称も並存しています。本記事では検索・比較文脈の実態に合わせて「Smartproxy (現 Decodo)」と表記し、以降は文脈に応じて Smartproxy または Decodo と書き分けます。

### 旧名称が並存している背景

- 既存の比較記事・コミュニティ議論が Smartproxy 名で蓄積されている
- 法人契約の請求書や契約書には旧名称が残っているケースがある
- X や Reddit では旧名称での投稿が依然として多い

### 検索時に注意すること

- レビュー記事は「Smartproxy 2024 年以前」と「Decodo 2025 年以降」で内容が分かれる
- 価格表は公式 Decodo サイトを最終的な真実とする
- 機能比較表で Smartproxy が並んでいる場合、Decodo と同じものを指しているか確認する

> 「Smartproxy のリブランド後 (Decodo) でも、クリーンな住宅 IP が安定しているという評価は変わっていない」(原文: Decodo (formerly Smartproxy) keeps the same reputation for clean residential IPs that holds up in real-world account management and iOS proxy setups.)

### 立ち位置の違い - エンタープライズ vs バリュー

両社は同じプロキシ市場にいながら、創業背景と訴求軸が異なります。**[Bright Data](https://get.brightdata.com/0cqcj8xp08fo) (旧 Luminati) は 2014 年創業、米国本社のエンタープライズリーダー** で、規模・SLA・コンプライアンスを軸に据えた契約形態を主力としています。一方 **Smartproxy は 2018 年創業、リトアニア発の中価格帯プロバイダ** で、2024 年に Decodo へリブランド。「個人〜中小規模で扱いやすい」「サブスク + 従量のハイブリッド料金」「ダッシュボードがシンプル」というポジションを明確に打ち出しています。

スペック面では Bright Data が 1.5 億超の Residential IP プール、195 か国の地理カバレッジ、Web Unlocker / Scraping Browser / Dataset Marketplace / MCP サーバーまで網羅する幅広いラインアップを持ち、料金は基本 Pay-as-you-go (PAYG) で月間ボリュームに応じたコミットメント割引が走る構造です。対する Smartproxy (Decodo) の Residential IP プールは 5,000 万〜7,000 万 IP 帯、地理カバレッジは主要 195 か国程度をサポートしますが、都市レベル細粒度や API ラインアップでは Bright Data に水を空けられています。「IP プール規模が直接 = 成功率」というわけではありませんが、ターゲットサイトが多様化したり同時セッション数が増えるほど Bright Data の余裕が効いてきます。

### AI / LLM 文脈での差

2026 年に入ってからの変化として、[Bright Data](https://get.brightdata.com/0cqcj8xp08fo) は LLM エージェントへの統合 (MCP サーバー経由で CAPTCHA や検知をエージェントが自動処理) を前面に出しており、AI スクレイピング基盤の選択肢として頭一つ抜けつつあります。

## 料金体系の比較 - 単価モデルの根本的な違い

両社の料金は単純な「単価の高い・安い」では語れません。**Bright Data は PAYG ベースの従量課金で、コミットメントで段階的に下がる構造**、**Smartproxy はサブスクリプション + 従量のハイブリッド** という根本的な違いがあります。料金体系の詳細は別記事の [Bright Data 料金プラン早見表 2026](/media/bright-data-pricing-2026) でも整理しているので、Bright Data 側の単価詳細はそちらも参照してください。

### 料金体系の根本的な違い

- **Bright Data**: PAYG が起点。最低契約金額なしで始められ、月間ボリュームに応じてコミットメント割引が走る
- **Smartproxy (Decodo)**: サブスクリプションプラン (例: $99/月で X GB 込み) を選び、超過分は従量。途中解約や下位プラン変更は柔軟

### 月間 50 GB の EC 価格モニタリングを想定したコスト試算例

具体的なボリュームで考えると、両社の単価設計の違いが見えやすくなります。月間 50 GB の EC 価格モニタリング (検知の厳しさは中程度、対象 5 サイト) を想定したシミュレーションは次のとおりです。

- **Smartproxy (Decodo) サブスク + 従量**: 月額プラン $200/月 (50 GB 込み) を基準ケースとした場合、超過 10 GB を従量 $4/GB で乗せると合計 $240/月 (≒約 38,000 円) 程度
- **Bright Data PAYG**: 単価 $6/GB × 50 GB で $300/月 (≒約 47,400 円)。月間 50 GB コミットでの $4.5/GB 帯 (要交渉) を取ると $225/月 (≒約 35,500 円)

ここまでは「Smartproxy がやや有利、Bright Data はコミット交渉で並ぶ」というレンジです。一方で Cloudflare / Akamai で守られたサイトを含めるなら、Bright Data の Web Unlocker (成功率ベース課金) を 1〜2 サイト分組み込んだ方が、Smartproxy で失敗リトライを重ねるより総コストが安くなるケースもあります。**「単価」ではなく「成功した 1 リクエストあたりの実質コスト」で比較するのが、契約後に後悔しないための判断基準** です。

### 失敗リクエストの再課金ポリシーと隠れコスト

成功した 1 リクエストあたりコストを正確に出すには、失敗リクエストの再課金ポリシーを把握しておく必要があります。Bright Data は Web Unlocker のように「成功時のみ課金」モデルを採用するプロダクトを揃えており、検知が厳しいサイトに対して総コストが読みやすい一方、Smartproxy (Decodo) は基本的に「リクエスト発生時点で帯域カウント」する従量モデルが中心です。検知が緩いサイトでは Smartproxy の単価優位が活きますが、検知の厳しい SNS や大手 EC で失敗率が 20〜30% を超えると、実質単価が PAYG 表示の 1.5 倍まで膨らみ得ます。契約前のトライアル段階で **失敗時の課金扱い・再試行ロジック・上限アラートを伝票化** しておくと、本契約後の請求書ショックを避けられます。

1. **月間 GB と対象サイトを PAYG / トライアルで実測する** (見積もりだけでは外れやすい)
2. **複数の対象サイトでテストする** (1 サイトだけだと偏る)

## 日本市場視点でのサポート差

日本市場で本番運用に乗せる際、料金単価以外で効いてくるのが「サポート言語」「請求書対応」「コンプライアンス対応」の 3 つです。ここの差は意外と大きく、稟議や法務レビューを通す段階で詰まることがあります。

### 日本語サポートと窓口体制

- **Bright Data**: 日本語のヘルプセンター、日本語問い合わせ窓口、日本語ドキュメントを整備。中堅以上の契約では日本担当の SE が PoC に伴走するケースもある
- **Smartproxy (Decodo)**: 英語中心。ダッシュボードの一部は日本語化されているが、サポートチケットは英語ベース。レスポンス速度自体は速い

### 請求書払いとインボイス対応

- **Bright Data**: 大口契約 (月額 $1,000 以上目安) ではインボイス払い・銀行振込・JPY 建て交渉が可能なケースあり
- **Smartproxy (Decodo)**: 原則クレジットカード / Wire Transfer。日本のインボイス制度対応は限定的

### コンプライアンス・法務面

- **Bright Data**: KYC ベースの倫理的 IP 取得、GDPR / CCPA / 個人情報保護法対応の社内体制、ISO 27001 取得、契約書のカスタマイズ余地が広い
- **Smartproxy (Decodo)**: 倫理的 IP 取得を公表、SOC 2 Type II 等の認証あり。契約書のカスタマイズ余地は限定的

### 稟議・法務レビューの典型的な詰まりどころ

実際に弊社が伴走してきた案件で詰まりやすかったポイントは次の 3 つです。

1. **個人情報保護法 (APPI) 関連の文書整備**: Bright Data は GDPR / CCPA / APPI を意識した DPA テンプレートが用意されており、法務確認が短縮しやすい。Smartproxy (Decodo) も DPA は用意されていますが、カスタマイズ余地は狭めです。
2. **インボイス登録番号・適格請求書対応**: 日本国内の経理プロセスで止まりやすい論点。Bright Data はインボイス制度に沿った請求書発行を相談ベースで対応するケースが増えています。Smartproxy 側はクレジットカード / Wire 中心で、適格請求書相当の書類を別途依頼する形になります。
3. **セキュリティ監査 (SOC 2 / ISO 27001) のエビデンス要求**: Bright Data は ISO 27001 を取得しており監査資料の提供が早い。Smartproxy も SOC 2 Type II 相当の認証を持ちますが、エビデンスの開示範囲は契約条件次第です。

### 日本市場で「使いやすい」判定

社内の経理プロセスや法務レビューを通す必要があるかどうかが分水嶺です。中堅以上で稟議を通したいなら Bright Data、個人開発や PoC レベルなら Smartproxy が現実解です。判断に迷う場合は、まず社内の購買・法務に「英語契約書 / クレジットカード請求 / SOC 2 のみ」で稟議が通る前提かを確認すると、選択肢が一気に絞り込めます。Residential と ISP の使い分け判断は [Bright Data Residential と ISP プロキシの使い分け実践ガイド 2026](/media/bright-data-residential-vs-isp) も参考にしてください。

## チーム規模別 - 選定フロー

ここまでの 5 軸を踏まえて、月間ボリューム・予算・コンプライアンス要件で分岐する選定フローを提示します。

> 「Bright Data, Smartproxy, Decodo はそれぞれ違う層をターゲットにしている。エンタープライズなら Bright Data、コスパ重視なら Smartproxy、Decodo は実利的な選択肢」(原文: Bright Data is enterprise-grade, Smartproxy emphasizes value and ease of use, and Decodo is a practical mid-tier option for clean residential IPs.)

<TweetCard id="2050272096693948913" />

### フロー 1: 月間 5〜30 GB の個人開発・小規模 PoC

- **推奨**: Smartproxy (現 Decodo)
- 単価が半額前後、サブスク + 従量で予算管理しやすい
- ダッシュボードがシンプルで初心者でも 30 分でセットアップ可能
- 対象サイトが Cloudflare / Akamai でなければ成功率も十分
- **想定ケース**: 個人開発の市場調査ツール、価格比較アプリのプロトタイプ、企業案件の予算 10 万円規模 PoC など。月額 $30〜100 のサブスクから始めて、上限アラートを契約初日に仕込んでおくのが定石です。

### フロー 2: 月間 30〜100 GB の中規模スクレイピング

- **推奨**: 両社をトライアル比較 → 成功率重視なら Bright Data、コスト重視なら Smartproxy
- 大手 EC を叩くなら Bright Data の Web Unlocker が刺さる
- SNS / 検索系で安定取得が必要なら Bright Data
- ニッチサイトや低検知サイトなら Smartproxy で十分
- **想定ケース**: 月間 50 GB の EC 価格モニタリング (上述シミュレーション)、SaaS 企業のマーケットインテリジェンス、新規事業のデータ仕入れ。両社 2 週間トライアルを並行で走らせ、成功率と実質単価で意思決定します。

### フロー 3: 月間 100 GB 以上・複数サイト・長期運用

- **推奨**: Bright Data 主軸
- IP プール規模・SLA・コンプライアンス監査体制でエンタープライズ要件を満たす
- コミット契約で Smartproxy との単価差が縮まる
- Web Unlocker / Scraping Browser / Dataset Marketplace を一気通貫で使える
- **想定ケース**: 検索エンジン結果の大規模監視、グローバル EC の競合価格追跡、AI モデル学習用の Web データセット構築。コミットメント割引と Web Unlocker の組み合わせで月額 $5,000〜$15,000 帯の事業基盤を作るレンジです。

### フロー 4: 日本国内法務レビュー・経理プロセス重視

- **推奨**: Bright Data
- 日本語サポート + インボイス対応 + 契約書カスタマイズで稟議を通しやすい
- KYC ベースの IP 取得で法務リスクを低減
- 個人情報保護法対応の社内体制が明文化されている
- **想定ケース**: 上場会社の新規事業、金融・保険・医療など法規制が厳しい業界、海外子会社との合算請求が必要なグローバル契約。「英語契約書だけでは社内承認が下りない」「インボイス登録番号入り請求書が必要」のいずれかが該当するなら Bright Data 一択です。

### フロー 5: AI / LLM エージェント連携

- **推奨**: Bright Data
- MCP サーバー経由でエージェントが CAPTCHA や検知を自動処理
- Web Unlocker / Scraping Browser が AI エージェント前提の API 設計
- Smartproxy も Scraper API を整備しているが、エージェント自動化深度では Bright Data がリード
- **想定ケース**: LLM ベースの調査エージェント、AI コンシェルジュサービス、社内ナレッジ管理の自動データ補強。エージェントが「失敗時に自己リトライ + 別経路探索」を回す前提なら、Bright Data の MCP サーバーが最短ルートです。

## 注意点と運用 Tips

両社を比較検討する際、現場で意外と詰まりやすい論点を整理します。

### 注意すべき論点

- **「無料トライアル」の条件は両社で違う**: Bright Data は試用期間と GB 制限つきの本格トライアル、Smartproxy は $1〜数ドルの低額トライアルが中心。条件の比較表をシートに残しておくと後工程で迷わない
- **失敗リクエストの再課金**: 両社で扱いが異なるため、PoC 段階で「成功時のみ課金プロダクトはどれか」を文書化しておく
- **PAYG 中の使いすぎリスク**: 上限アラートと予算キャップは契約初日に設定しておく。Slack / メール通知の閾値も明示的に設定する
- **同時セッション数の制限**: 大規模並列実行時に効いてくる。Bright Data は Zone 設計で上限を分散できるが、Smartproxy はプラン段階で上限が決まっているケースがある
- **コミット契約後のダウングレード制限**: 解約 / 縮小条件は契約書の細部に注意。クォーター単位の調整が可能か事前に確認する
- **個人情報を含むスクレイピングの法務確認**: 公開情報であってもメールアドレス・氏名を含むケースは個人情報保護法の射程に入りやすいため、データ取得目的と保管期間を明文化しておく

<Callout type="warning" title="注意">
本記事の数値は 2026 年 5 月時点の公開情報と弊社の運用実感をもとにしています。プロキシ業界は料金変更やプロダクト改廃が速いので、契約前には両社の公式ページで最新の料金・SLA をご確認ください。また「検知されない」「常時成功する」と断定する業者は実態と乖離しているケースが多く、現実的には目標成功率 (例: 95% 以上) を SLA や測定値で確認するのが妥当です。スクレイピング対象サイトの利用規約と robots.txt の遵守、レート制御、個人情報の取り扱いには十分にご注意ください。
</Callout>

## まとめ

Bright Data と Smartproxy (現 Decodo) は同じ Residential プロキシ市場にいながら、ターゲット層が明確に異なります。[Bright Data](https://get.brightdata.com/0cqcj8xp08fo) は「規模・コンプライアンス・自動化ツール・日本市場対応」を強みとするエンタープライズリーダー、Smartproxy は「コスパ・使いやすさ・サブスク + 従量の柔軟性」を強みとする中価格帯バリュープロバイダです。月間 30 GB 以下なら Smartproxy が現実解、それ以上か日本国内法務レビューを通したいなら Bright Data という分け方が 2026 年時点の実務的な判断軸となります。本契約前には両社の無料 / 低額トライアルで対象サイトを 1〜2 日叩いて成功率と単価を実測することをおすすめします。机上の単価比較だけで決め切らず、実測の数字で稟議資料を作るのが結果的に最短ルートです。
