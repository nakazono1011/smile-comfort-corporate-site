「1Password と Bitwarden、結局どちらを選ぶべきか」というご相談を頂きます。本記事では 2026 年 5 月時点の公式情報をベースに、料金・セキュリティ・機能・運用負荷の 4 軸で [1Password](https://1password.partnerlinks.io/sc-link) と Bitwarden を比較し、個人ユース・家族共有・小規模チーム・社内導入のシーン別にどちらが合うかを整理します。中立性を保ちつつ、判断の材料を公式ソースの脚注付きで提示します[^1][^2]。

## 1Password と Bitwarden の立ち位置 (2026 年 5 月時点)

両者ともゼロ知識暗号化 (サーバ側でも復号できない設計) のパスワードマネージャーで、AES-256、Passkey、自動入力、セキュアノート、ブラウザ拡張、各種 OS クライアントを揃えています[^1][^2]。安全性そのものに優劣はほぼなく、差は **「商用 SaaS の完成度」 vs 「OSS の自由度」** という設計思想の違いに集約されます。

### 1Password の立ち位置

- **商用 SaaS のリーダー**: UI / UX の洗練度、Watchtower (漏洩監視) や Travel Mode (国境通過時の Vault 隠し) のような細部の機能が成熟[^1]
- **エンタープライズ向け機能が厚い**: SSO 連携 (Okta / Azure AD / Google Workspace)、SCIM プロビジョニング、Secrets Automation (CLI / CI/CD 統合) が標準[^5]
- **クローズドソース** だが、第三者監査 (SOC 2 Type II, ISO 27001) を継続的に公開[^6]
- **本社**: カナダ・トロント。サポート言語は英語中心 (日本語ヘルプは限定的)

### Bitwarden の立ち位置

- **OSS のチャンピオン**: クライアント・サーバの両方を AGPL v3 で公開、コード監査が誰でも可能[^2]
- **無料プランが強力**: パスワード無制限・端末無制限の Free プランがある (個人ユースで「とにかく無料で始めたい」需要を取り込み)[^2]
- **セルフホスト可能**: 公式の Docker イメージで自前運用が可能。コミュニティ実装の Vaultwarden (Rust 実装) で軽量化も可能[^3]
- **本社**: 米国・フロリダ。Free → Premium ($10/年) → Families ($40/年) の階段がシンプル

### X 上の温度感

2026 年に入ってからの両者の価格改定 (1Password の年額 $12 値上げ、Bitwarden Premium の段階的引き上げ) によって、SNS では「OSS / セルフホスト」志向の議論が盛り上がっています。

> 「両者とも安全。違いは UX とサポートの厚さ vs OSS と価格の透明性。どちらを取るかは『時間を買うか、自由を取るか』の問題だ」(原文の主旨: Both are safe; UX vs OSS, time vs freedom.)

## 料金比較 — 個人・家族・チーム

公式の年払いベースで並べると以下のようになります[^1][^2]。レートは執筆時点の USD で記載しています。

| プラン | 1Password | Bitwarden | 補足 |
|---|---|---|---|
| 無料 | なし | **Free** (無制限パスワード・無制限端末) | Bitwarden Free は事実上の主力 |
| 個人 | Individual **$47.88/年** ($3.99/月) | Premium **$19.80/年** ($1.65/月) | 1Password は Watchtower 込み |
| 家族 | Families **$71.88/年** ($5.99/月、最大 5 名) | Families **$47.88/年** ($3.99/月、最大 6 名) | 家族プランは Bitwarden が安価 |
| 小規模チーム | Teams Starter Pack **$19.95/月固定** (10 名まで) | Teams **$4/月/ユーザー** | Bitwarden は人数連動、1Password は固定 |
| ビジネス | Business **$7.99/月/ユーザー** | Enterprise **$6/月/ユーザー** | 大規模では Bitwarden が微差で安い |

1Password の料金詳細 (年払い・月払いの差や SourceNext 3 年版との比較) は [1Password 個人 vs ファミリープラン徹底比較](/media/1password-pricing-individual-vs-family) でも整理しています。

### 「安さ」を額面通り受け取らない

Bitwarden Free は確かに無料ですが、TOTP 保存・緊急アクセス・ファイル添付・高度なセキュリティレポートを使うには Premium に上げる必要があります[^2]。家族で複数人運用するなら Premium × 2 名分でも $39.60/年、Families ($47.88/年、最大 6 名) のほうが 1 人あたりが下がるため、世帯人数が多ければ家族プラン一択になります。

> 「Bitwarden Premium は元々の年 $10 から段階的に値上がりした。それでも 1Password よりは安く、機能差は許容範囲」(原文の主旨: Bitwarden Premium rose from $10/year but is still much cheaper than 1Password.)

### 法人での実コスト

中小〜中堅企業 (10〜100 名規模) で計算すると、年間ライセンス費の差は 1 人あたり数百円〜千円程度に収まります。100 名で年間 10 万円前後の差ですが、SSO 連携工数 / 緊急時のサポート対応 / 監査ログ運用にかかる人件費の差を考えると、額面の差を運用工数差で吸収できるかが判断軸になります。

## セキュリティ・コンプライアンスの比較

### 暗号化アーキテクチャ

| 項目 | 1Password | Bitwarden |
|---|---|---|
| 暗号化方式 | AES-256-GCM | AES-256-CBC |
| 鍵導出 | PBKDF2-HMAC-SHA256 (650,000 イテレーション) | PBKDF2 or **Argon2id** (推奨) |
| マスターパスワード以外の保護 | **Secret Key (128bit)** が端末側に保管され、サーバ漏洩でも復号不能 | マスターパスワードのみ (Argon2id 強度に依存) |
| ゼロ知識 | ◯ | ◯ |
| ソース公開 | クライアント / サーバ非公開 (独立監査レポートを公開)[^6] | **クライアント / サーバ AGPL v3 で完全公開**[^2] |

[1Password](https://1password.partnerlinks.io/sc-link) の Secret Key は「マスターパスワードが流出した場合でも、サーバ側のデータだけでは復号できない」という点で、攻撃ベクトルが 1 つ多い分だけ堅牢性が高い設計です[^5]。逆に Bitwarden は **コードを誰でも検証できる** という意味での透明性で勝ります。脅威モデルとして「サーバ事業者を信用しない」を重視するなら Bitwarden + Argon2id、「攻撃者が認証情報を窃取するシナリオを防ぐ」を重視するなら 1Password の Secret Key 構造、というように使い分けるのが現実的です。

### 監査・準拠

両者とも以下を継続的に取得・公開しています[^5][^6]。

- SOC 2 Type II
- ISO 27001 / 27017 / 27018
- GDPR / CCPA / HIPAA (運用上の準拠)
- 年次の第三者ペネトレーションテスト報告書

差はほぼなく、いずれも法人導入で要求される標準的なコンプライアンス要件は満たします。

## 機能・運用面の比較

### 共通機能 (横並び)

両者とも以下を備えており、機能の有無で差が出る場面は限定的です[^1][^2]。

- パスワード生成 (長さ・記号・覚えやすさのカスタム)
- TOTP (2 要素認証コード) の保管
- Passkey (FIDO2 / WebAuthn) 対応
- 自動入力 (ブラウザ・モバイル)
- セキュアノート / クレジットカード / 個人情報の保管
- 各 OS クライアント (macOS / Windows / Linux / iOS / Android / Chrome / Safari / Firefox)
- CLI (`op` for 1Password, `bw` for Bitwarden)

### 差別化機能 (どちらか片方にしかない)

| 機能 | 1Password | Bitwarden |
|---|---|---|
| Watchtower (漏洩監視) | ◯ | 一部 (Premium のみ Have I Been Pwned 連携) |
| Travel Mode (Vault 隠し) | ◯ | × |
| SSH キー管理 + Git 署名 | ◯ (`op` CLI) | × |
| Secrets Automation (CI/CD 統合) | ◯ | △ (Bitwarden Secrets Manager 別契約) |
| セルフホスト | × | **◯ (公式 + Vaultwarden)** |
| OSS クライアント | × | **◯ (全 OS)** |
| Send (一時共有リンク) | △ (Item Sharing) | ◯ (テキスト・ファイル両対応) |

[1Password の Passkey 活用ガイド](/media/1password-passkeys-guide) では SSH キー管理や Passkey 関連機能の運用設計を別途整理しています。

> 「1Password の UX は群を抜いてる。家族 5 人で共有保管庫を回すなら 1Password が一番ストレスがなかった」(原文の主旨: 1Password's family sharing UX is the smoothest.)

### 運用負荷

- **1Password**: SaaS のみ。サーバ運用が不要。サポートチームへの依存度は高いが、ダウンタイムや TLS 証明書を意識する必要なし
- **Bitwarden (公式 SaaS)**: 同上。Free プランの存在で「お試し→有償移行」がスムーズ
- **Vaultwarden (セルフホスト)**: VPS 代 $5〜23/月で運用可能。ただしバックアップ・TLS・OS パッチ・侵入検知を自社で抱える。社内 SRE / 情シス体制がない場合は推奨しない[^3]

## ユースケース別の選び方

### 個人ユース (1 人で完結)

- **コストを最優先 / 機能は最低限で OK** → Bitwarden Free
- **TOTP 保管・緊急アクセス・添付ファイルが必要** → Bitwarden Premium ($19.80/年)
- **Watchtower や Travel Mode、UX を重視** → 1Password Individual ($47.88/年)
- **3 年単位で固定したい・為替リスクを切りたい (日本)** → SourceNext 経由の 1Password 3 年版

### 家族・パートナー共有 (2〜6 人)

- **コスト最優先 + Bitwarden の UI に慣れている** → Bitwarden Families ($47.88/年、6 名まで)
- **UX・サポートに投資、Passkey や Watchtower を全員で活用** → [1Password Families](https://1password.partnerlinks.io/sc-link) ($71.88/年、5 名まで)

### 小規模チーム (5〜30 名)

- **OSS と自由度、SCIM 不要、人数が増減する想定** → Bitwarden Teams ($4/月/ユーザー)
- **固定費 + 安定運用、SSO は将来検討** → 1Password Teams Starter Pack ($19.95/月固定、10 名まで)

### 中規模以上の社内導入 (30 名以上)

- **SSO・SCIM・監査ログ・Secrets Automation を全部使いたい** → [1Password Business](https://1password.partnerlinks.io/sc-link)。Business プランの導入は [1Password Business プランの選び方](/media/1password-business-pricing) でも整理しています
- **OSS と自前運用、コード監査の透明性を重視** → Bitwarden Enterprise + Vaultwarden ハイブリッド

## 乗り換えの実務 — Bitwarden → 1Password / 1Password → Bitwarden

### Bitwarden → 1Password

1. Bitwarden の Web Vault から JSON エクスポート (パスワード + TOTP + カスタムフィールド)
2. 1Password の Import で「Bitwarden」を選択 → JSON をアップロード
3. TOTP シードキーの整合を確認 (一部の TOTP は再登録が必要)
4. 添付ファイルは個別ダウンロード → 1Password 側で再添付

公式の [1Password 引っ越しサポート](https://1password.partnerlinks.io/6dieu4x28dzi-gp0g2q) を利用すると、Bitwarden の未消化サブスク期間を 1Password 側が一部負担してくれる場合があります[^4]。乗り換え時の契約重複コストを気にせず移行できるので、検討中の方は申請してみる価値があります。

### 1Password → Bitwarden

1. 1Password から 1PUX (1Password Unified Export) をダウンロード
2. Bitwarden に「1Password 1PUX」インポーターでアップロード
3. Vault が複数ある場合は Bitwarden の Organization / Collections に再マッピング
4. 共有 Vault は Organization の Owner 権限で再構築

両方向とも所要は 30 分〜2 時間程度。データ量が多い場合は事前に整理しておくと工数が減ります。

## まとめ — 判断軸の整理

最後に判断軸をまとめます。

- **時間で課金する (UX とサポートに投資)** → 1Password (Individual / Families / Business)
- **自由度と OSS で課金する (人月とコスト最適化)** → Bitwarden (Free / Premium / Enterprise + Vaultwarden)
- **個人で迷ったら**: Bitwarden Free で 1 か月使い、足りなければ 1Password Individual に上げる二段階アプローチ
- **家族・チームで迷ったら**: UX と Passkey 運用を重視するなら 1Password、コストと OSS 透明性なら Bitwarden
- **乗り換え検討**: 1Password の引っ越しサポートで既存ライセンス費用の一部負担を受けられる可能性あり[^4]

両者とも安全性は同等です。「正解はあなたの組織がサーバ運用に投資できるかどうか」で決まります。迷ったら、まず 30 日間の無料トライアルで 1Password の UX を確認し、肌に合わなければ Bitwarden Free に乗り換える、という形で実際の運用感を確かめるのが手堅い進め方です。
