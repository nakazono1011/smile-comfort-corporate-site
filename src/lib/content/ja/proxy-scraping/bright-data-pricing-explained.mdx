# Bright Data の料金体系完全解説

## はじめに

Bright Data（旧 Luminati）は世界最大級のプロキシネットワークを提供する企業ですが、その料金体系は複雑で初心者には理解しにくいことで知られています。本記事では、Bright Data の全料金プランを詳しく解説し、あなたのビジネスに最適な選択肢を見つけるためのガイドを提供します。

## Bright Data の料金体系概要

Bright Data の料金は主に以下の要素で構成されています：

- **プロキシタイプ別料金**
- **使用量（GB）による従量課金**
- **同時接続数制限**
- **地域・国別プレミアム**
- **企業向け追加機能**

## 従量課金制の詳細

### 基本料金構造

Bright Data の従量課金制では、以下の要素が料金に影響します：

```
総料金 = 基本月額料金 + 超過使用料 + 地域プレミアム + 機能追加料金
```

### 使用量計算方法

**データ使用量の計算：**

- リクエスト送信時のデータ
- レスポンス受信時のデータ
- HTTP ヘッダーのサイズ
- 失敗したリクエストも課金対象

**例：1MB のページを 1000 回取得する場合**

```
データ使用量 = 1MB × 1000回 = 1GB
料金 = 1GB × GB単価（プランによる）
```

```python
import datetime

class UsageMonitor:
    def __init__(self):
        self.daily_usage = {}

    def track_request(self, size_bytes):
        today = datetime.date.today()
        if today not in self.daily_usage:
            self.daily_usage[today] = 0

        self.daily_usage[today] += size_bytes / (1024**3)  # GB変換

    def get_monthly_projection(self):
        avg_daily = sum(self.daily_usage.values()) / len(self.daily_usage)
        return avg_daily * 30
```
